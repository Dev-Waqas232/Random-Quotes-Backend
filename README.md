# RanGen ( Backend )

A backend for a random quote generator API, developed using FastAPI. The main purpose of this project is to learn about fundamentals of FastAPI.

## Links

- [Live URL](https://random-quotes-backend-t4ol.onrender.com)
- [Documentation](https://random-quotes-backend-t4ol.onrender.com/docs)

## Goals Achieved

- [x] Python modules, use of `___init___` files
- [x] GET API routes
- [x] Handling environment variables in python
- [x] Error Handling in FastAPI
- [x] Using Third Party APIs
